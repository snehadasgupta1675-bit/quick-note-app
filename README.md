# Quick Note App 📝

A simple Full Stack web application where users can create and save notes.

## 🚀 Features
- Add notes
- View saved notes
- Data stored in MongoDB
- Full Stack project (Frontend + Backend)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

## 📌 How it works
User enters a note → clicks Save → backend stores it in MongoDB → frontend displays saved notes.
